股市Coding｜隱私權政策

最後更新日期：2025/10/10  ·  ← 返回首頁

本政策說明 股市Coding（以下稱「本服務」）如何蒐集、使用、保存與保護您的資料，以及您可如何行使相關權利。使用本服務即表示您同意依本政策進行資料處理。

一、我們蒐集哪些資料

  • 訂閱與授權資訊（平台提供）：為完成購買／訂閱驗證與權益（Entitlement）判定，本服務依照各平台（Google Play（Android） 與 App Store（iOS））規範處理必要資訊，例如：訂閱有效狀態、到期時間、交易識別碼等。
  • 裝置與診斷資料：為提升效能與穩定性，可能蒐集匿名化或去識別化之裝置資訊（如裝置型號、OS 版本、App 版本）與錯誤紀錄（Crash/Log），僅用於偵錯與品質優化。
  • 使用情境資料（可選）：在必要時程中可能進行使用趨勢與功能成效統計；此類資料以聚合或去識別化方式處理。

資料最小化原則：我們僅處理完成訂閱驗證、服務運作與安全所需之必要資訊，不涉入與上述目的無關之個人資料。

二、我們如何使用資料

  • 訂閱驗證、權益判定與功能啟用。
  • 服務維運、效能優化、錯誤偵測與資安防護（含反濫用與公平性維護）。
  • 提供必要通知（例如重大變更或中斷公告）。
  • 法令遵循與爭議處理。

三、第三方服務與平台

本服務可能整合或使用第三方平台或服務（例如 Google Play、App Store、分析與雲端服務）以完成訂閱驗證、付款處理、錯誤診斷或使用統計。第三方對資料之處理，適用其各自的隱私權政策。

四、資料保存期間

我們會於達成本政策所述目的之必要期間內保存資料，並可能因法令遵循、爭議處理或安全需求而適度延長。保存期滿或目的消失後，將依適用法令與內部程序刪除或去識別化處理。

五、如何刪除或移除資料／取消授權（Android／iOS 通用）

  • Android（Google Play）：於 Google 帳號或 Play 商店相關設定中管理與本服務相關之訂閱與資料；如需刪除 Play Games 或應用資料，請依 Google 提供之帳號與隱私設定操作。
  • iOS（App Store / iCloud）：於 iOS 設定管理您的訂閱；若資料存於 iCloud，請至 設定 → Apple ID → iCloud → 管理儲存空間 操作刪除；如有 Apple 帳號授權連結，請於 設定 → Apple ID → 密碼與安全性 → 使用 Apple ID 的 App 檢視與調整。
  • 聯繫協助：如需協助進行資料刪除或取消授權，請透過下方聯絡方式與我們聯繫，我們將依身分確認與適用法令協助處理。

六、資料分享與揭露

  • 服務供應商：於必要範圍內委託處理（例如雲端、分析、錯誤回報），並以契約或等同機制要求其遵循資料安全義務。
  • 法令要求：於法定義務、司法或主管機關合法請求之情形下，依法揭露必要資訊。
  • 權利保護：為保護本服務、使用者或公眾之權益、安全與財產，於必要時得為適當之揭露。
  • 交易或重組：於合併、收購或資產交易過程中，可能在合理必要範圍內處理或移轉相關資料；並於實體變更後持續提供相應保護。

七、資料安全與跨境傳輸

我們採取合理之技術與管理措施以保護資料安全；然網路傳輸與電子儲存並非絕對安全。我們之服務與供應商可能在不同法域運作，使用本服務即表示您同意資料於必要範圍內進行跨境處理與保存。

八、未成年人保護

本服務非以 13 歲以下兒童為對象。若得知未經監護人同意而蒐集至該等資料，將採取合理措施刪除或停止處理。監護人若有疑慮，請與我們聯繫。

九、您的權利

依適用法令，您得就個人資料行使查詢閱覽、製給副本、更正、停止處理或利用、刪除等權利；我們可能需要先行進行身分確認，並依法定期間回覆或處理。

十、政策更新

我們可能不時更新本政策；重大變更將於本頁公告並更新「最後更新日期」。建議您定期查閱最新版本。
