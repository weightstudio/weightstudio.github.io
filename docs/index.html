應用簡介

專為 短線波段投資者 打造的大數據回測與訊號分析系統。每日收盤後自動掃描上億訊號點，將多種技術指標進行統計與交叉驗證，提供具體且可操作的參考名單。

核心功能

  • 歷史回測｜自動解析歷史資料，評估訊號在不同市場環境下的表現。
  • 五大指標｜勝率、獲利、持有天數、發生次數、標準偏差，一眼掌握報酬與風險。
  • 動態排名｜每日更新，快速鎖定高參考價值標的。
  • 自動偵測｜追蹤訊號延續性，協助掌握持續性機會。

四大特色與優勢

  • 多指標綜合分析：避免單一指標失準風險，提升穩健度。
  • 安全評分保障：內建穩定性評分，過濾雜訊更放心。
  • 效率導向介面：重點資訊即時呈現，決策更有效率。
  • 新手友善：簡潔操作流程，入門快速上手。
